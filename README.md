CSS Selectors & Specificity

Навчальний проєкт, що демонструє роботу CSS селекторів та їх специфічність.

📌 Опис

Проєкт показує, як різні типи селекторів впливають на застосування стилів у HTML.
Кожен елемент на сторінці має підпис із вагою селектора, що дозволяє наочно зрозуміти пріоритети CSS.

🎯 Мета
Зрозуміти різницю між селекторами
Вивчити специфічність (specificity)
Побачити, як CSS вирішує конфлікти стилів
🧩 Використані селектори
Тег (p, li, a)
Клас (.box1, .container)
ID (#unique)
Атрибут ([type="text"])
Вкладені (div span)
Дочірні (section > p)
Сусідні (h2 + p)
Комбіновані (div.combo p)
⚖️ Специфічність

Пріоритет селекторів:

ID — 100
Class / Attribute — 10
Tag — 1

При конфлікті стилів застосовується селектор з більшою специфічністю.

🛠 Технології
HTML5
CSS3
📂 Структура проєкту
index.html — приклади селекторів
style.css — стилі та демонстрація специфічності
🚀 Запуск

Просто відкрий index.html у браузері.
